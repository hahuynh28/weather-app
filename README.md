# Weather App

A simple weather application built with React, focusing on API integration,
state management, and clean component-based architecture.

## Features

- Search current weather by city name
- Loading, empty, and error states
- Persist last searched city using localStorage
- Clear separation between data logic and UI components

## Tech Stack

- React
- Vite
- Tailwind CSS
- shadcn/ui
- OpenWeather API

## Getting Started

```bash
npm install
npm run dev
